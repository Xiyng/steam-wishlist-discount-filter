Steam Wishlist Price Filter

Steam Wishlist Price Filter is a userscript that adds the option to filter games by price or show only discounted items in your Steam wishlist.

  • Download
  • GitHub repository

Features

  • Filter games by price
  • Show only discounted items on your Steam wishlist
  • Choose shown items by discount percentage
